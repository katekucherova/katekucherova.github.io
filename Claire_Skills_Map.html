Power BI / DAX
Safe Division & Null-Guarded AggregationDIVIDE · ISBLANK · COALESCE
Every ratio (NOI Margin %, Variance %) uses DIVIDE with an implicit zero-guard, and every budget comparison checks ISBLANK before subtracting — no #DIV/0! or blank-row errors reach the report.
Filter Context ControlCALCULATE · REMOVEFILTERS · ALLEXCEPT · SELECTEDVALUE · VALUES
Seasonality Index recalculates each month's revenue against its own year's average using ALLEXCEPT; Payment Mix Insight uses REMOVEFILTERS to compare one program against the whole table regardless of what's currently sliced.
Cross-Table LookupsLOOKUPVALUE
Break-Even Target, Incremental Fixed Cost, and Contribution Margin all pull from a separate assumptions table by name, rather than hardcoding a value into the visual layer.
Dynamic Narrative GenerationVAR/RETURN · FORMAT · SWITCH
Payment Mix Insight assembles a full written sentence conditionally from live data ("Karate had 12% of transactions classified as one-time…") — the report writes its own commentary as the data changes, rather than showing a static caption.
